# Luminous AI 🌟

> *Think brighter. Chat smarter.*

Luminous is a beautifully designed AI chatbot mobile app that brings clarity and intelligence to every conversation.

---

## ✨ Features

- 🤖 AI-powered intelligent conversations
- 📱 Clean & modern mobile UI
- ⚡ Fast & real-time responses
- 🌙 Dark & light mode support
- 🔒 Secure & private

---

## 🚀 Getting Started
```bash
# Clone the repository
git clone https://github.com/yourusername/luminous.git

# Navigate to project
cd luminous

# Install dependencies
npm install

# Run the app
npm start
```
